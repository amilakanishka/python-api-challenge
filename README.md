# python-api-challenge

# Resource files
Solution file is WeatherPy.ipynb. In order to execute the solution, api_keys.py file needs to be placesed which contains two keys weather_api_key and g_key. "api_keys.py" file needs to be placed inside both folders WeatherPy and VacationPy.
Output figures and csv is getting saved to python-api-challenge\WeatherPy\output_data folder.

## Installation

Solution is in python-api-challenge\VacationPy and python-api-challenge\WeatherPy. Please open the solution file from Jupyter Notebook
